Supporting code for resource bundle load strategies.

The framework for resource bundle load strategies is defined in {@link org.ppwcode.vernacular.l10n_III.resourcebundle.ResourceBundleLoadStrategy}.

Some specific implementations of this framework are provided in this package too.

Finally, the package offers static helper methods that aid in using resource bundles. These methods use a {@link org.ppwcode.vernacular.l10n_III.resourcebundle.ResourceBundleLoadStrategy}.
